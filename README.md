# 개인 로그북 시스템

나의 하루를 구조화하여 기록하고 성찰하는 웹 기반 로그북입니다.

## 개발 목적
일상과 업무의 기록을 체계화하여 지속적인 성찰과 성장을 지원

## 주요 기능
- 일자별 기록 (오늘 할 일, 주요 활동, 메모, 성찰, 내일 할 일)
- 건강관리 체크박스
- 임시저장 (로컬스토리지)
- 이전/다음 기록 조회
- Google Sheets 자동 저장

## 기술 스택
- Frontend: HTML / CSS / JavaScript
- 저장소: Google Sheets
- 연동: Google Apps Script
- 배포: GitHub Pages
